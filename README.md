# PixelArt

Un semplice programma per disegnare pixel art, utilizzabile online o scaricabile e aperto con un browser.

## Link

- **GitHub Pages:** https://vroby65.github.io/pixelart/
- **TinyURL:** https://tinyurl.com/pixelart65

## Funzionalità

- Griglia di disegno con risoluzione regolabile (8×8, 10×10, 16×16, 32×32, 64×64)
- Palette colori con gradazioni e sfumature
- Selezione colore primario (click sinistro) e secondario (click destro)
- Salvataggio e caricamento progetti in formato `.pixelart`
- Esportazione in PNG con trasparenza sui pixel neri
- Stampa dell'immagine
- Analisi delle sequenze di colore per riga
- Schermo intero
